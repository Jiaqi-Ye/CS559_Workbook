Z-Fighting: Co-Planar Face Decals

Each cube face has a second polygon lying on (or just above) the same depth as the base face. At zero offset, depth-buffer precision causes unstable draw order and shimmering.

Separation Control
0.0000
Try This
1) Set height to 0.0000 and rotate the cube. The orange decals will flicker against the blue faces.

2) Increase separation slowly. A tiny offset should make the decals stable and always visible.
Navigation
Left-drag: orbit camera
Scroll: zoom
