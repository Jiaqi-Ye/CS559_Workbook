Page 20: Material Property Maps

Up until now, we’ve only been using texture to change colors. However, we can use a texture to control just about any property of a material. If you look at the documentation for MeshStandardMaterial, you can see that many of the properties we control (such as roughness, emission, and metalness) can also be controlled with a map, rather than by providing a single value for the entire object.

Here’s a simple example: I load a checkerboard texture and use it to modify the roughness parameter. So some parts are rough, and other parts are shiny.

demo-physical-roughness-map.html

All this takes is assigning the texture to the map property. Notice that I still need to set the “base” roughness (since the map value and the base value are multiplied together).

In Three.js, the roughness map uses the green channel of the texture. This specific channel mapping is built into the Three.js lighting models to support the industry-standard ORM texture packing method. In PBR workflows, artists frequently pack three different black-and-white data maps into a single RGB image file to save memory and performance:

  • Occlusion (aoMap) -> Red Channel
  • Roughness (roughnessMap) -> Green Channel
  • Metalness (metalnessMap) -> Blue Channel

Changes in material properties often work better when they are combined with other kinds of maps (like color).

For this exercise, pick one (or more) properties, and make a map that controls the property over the object. You need to create a demonstration where the effect is obvious. This means creating a texture with variation, having an appropriate object to show (a primitive like a sphere is often good enough), and maybe some adjustments to the lighting. Try using the property realistically, for example, making the metal parts of an object more “metal” than the rest.

Note: To show that you haven’t just changed the colors, you probably want to make either the light or the object move (with animation). But you should know how to do that from previous workbooks.

To show that you really understand the lighting effects and how they are influenced by the material properties (which are, in turn, specified by the map), in the text box, explain what material property you are altering, and how this varying property can be seen in the resulting pictures. Be specific (tell us what to look for, and why it is there).

You will probably want to create your own texture map. Make sure you include it in the repository.
