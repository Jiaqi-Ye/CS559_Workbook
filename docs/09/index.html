Page 9: Cubics

We will create complicated shapes by putting together simple pieces. We need to do this carefully; otherwise, we won’t get continuity.

If we try to make shapes out of linear pieces (line segments), we can never do better than C(0) continuity (except for a straight line). The common practice in computer graphics is to use cubic polynomial segments. That will allow us to get C(1) (or better) continuity, and also have a lot of flexibility in the shapes that we can create.

We could use even higher-degree polynomials (quartic polynomials, quintic polynomials, etc.). However, for many reasons, in computer graphics, cubics are usually sufficient.

Piecewise Polynomials

We will be making our curves by breaking a large curve into a series of smaller segments, and representing each segment as a polynomial. Our curves are piecewise polynomials because each piece is a polynomial. Mathematicians use the term spline to refer to piecewise polynomials; however, this term sometimes has other connotations.

Before considering the pieces, we should be clear about how they fit together. For the moment, denote the overall curve as f∗​(t) for t in some range. We construct this curve from a list of segments that are also curves: f0​(t), f1​(t), etc. Each of the segments applies to some portion of the “overall time,” so we could do something like:

f∗​(t)=if elif elif etc.​0≤t<1 then f0​(t)1≤t<2 then f1​(t)2≤t<3 then f2​(t)​

for however many pieces we might have, with the slight caveat that we need to deal with closing the interval at the end (so fn​ applies over the range n≤t≤n+1).

This is somewhat inconvenient, since it means each segment must be defined in terms of where it goes in the overall time. It is much better (and more common) to define each segment as spanning the range 0≤u<1 (or ≤1 if appropriate), so:

f∗​(t)=if elif elif etc.​0≤t<1 then f0​(t−0)1≤t<2 then f1​(t−1)2≤t<3 then f2​(t−2)​

Of course, we could choose to make the “breakpoints” where we switch functions occur at other times, and shift and scale time accordingly so that each fi​ gets the appropriate range.

Since each of our pieces is a polynomial, we know that within each piece we have continuity (for all derivatives). Therefore, we only need to be concerned about continuity where the functions switch. If we want C(0) continuity, we need to make sure fi​(1)=fi+1​(0) (the end of one piece equals the beginning of the next). If we want C(1) continuity, we need to make sure that fi′​(1)=fi+1′​(0) (the first derivative at the end of one segment is equal to the first derivative at the beginning of the next).

Polynomials

A polynomial is a function of a single parameter (in this case t):

f(t)=a0​+a1​t+a2​t2+⋯+ad​td
f(t)=i=0∑d​ai​ti

The ai​ are sometimes referred to as coefficients.

A few simple observations:

  1. If we have a polynomial of degree d (that is, the highest power of the polynomial is d, so there is a td term), we have d+1 coefficients.
  2. We can write the coefficients as a vector of length d+1.
  3. We can think of a quadratic (a degree-2 polynomial) as a cubic (a degree-3 polynomial) with its a3​ coefficient as 0.
  4. While the equation is non-linear in the parameter, it is linear in the coefficients. That is, for a given value of the parameter t, f(t) is a linear function of the coefficients ai​.

There are many different ways to write a polynomial. The “coefficient form” above is just one way to write it. For example, if we have a cubic polynomial with the coefficient vector a=(−6,11,−6,1), which gives us f(t)=−6+11t−6t2+t3 (we put a0​ first in the vector), we could also put it in the form (t−1)(t−2)(t−3).

This idea of having different forms of polynomials will be important. Ultimately, they are all the same polynomial, but some forms will be easier to work with than others.

Cubics

A cubic is a polynomial of degree 3.

We write a cubic polynomial (in coefficient or canonical form) as:

f(u)=a0​+a1​u+a2​u2+a3​u3

denoting the coefficients as a0​,a1​,a2​,a3​ - in boldface since they are vectors. This is a nicer notation than writing (if our vectors are 2D):

fx​(u)=a0x​+a1x​u+a2x​u2+a3x​u3 fy​(u)=a0y​+a1y​u+a2y​u2+a3y​u3

It’s useful to remember that we have a separate function for each dimension (fx​ for the x-coordinate and fy​ for the y-coordinate).

Notice that a cubic segment has 4 coefficients. When we specify the cubic in some other way, we need to give enough information to compute these four values. If we give too little information or too much, it won’t work exactly. Therefore, other representations will use 4 numbers too.

Trying to control the cubic by specifying the coefficients is not easy (e.g., what values for a0​,a1​,a2​, and a3​ give a desired shape?). Therefore, we almost always specify cubic curves using some other form.

If we want to connect cubics, we would like to specify the beginning (where u=0) and end (where u=1). Let’s call these points p0​ and p1​. If we write things in canonical form, the beginning is straightforward:

p0​=f(0)=a0​

The end is a bit more complicated.

p1​=f(1)=a0​+a1​+a2​+a3​

The problem is that we have many choices, and this doesn’t say what happens in between the endpoints.

The solution will be to use different forms for the cubics. We’ll specify values that are convenient, and compute the coefficients from these. Alternatively, we can think of these different representations and simply use the polynomials in these other forms. As an analogy, if I ask you to compute f(3) for the cubic polynomial in the last section, you can compute it by plugging the number into the factored form f(t)=(t−1)(t−2)(t−3) without figuring out what the coefficients are.
