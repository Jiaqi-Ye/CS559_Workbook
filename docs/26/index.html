Page 26: Student's Choice!

This worked well on Workbook 1, so we’ll try it again…

Student’s Choice! Here’s a big empty box. Make something cool that uses curves. It should (1) have some curved shapes; (2) have some object that follows a curved path; and (3) be cool. Your program must be “animated”; things must move even if there is no user interaction. You can have interaction (be sure to give instructions), but it is not required.
